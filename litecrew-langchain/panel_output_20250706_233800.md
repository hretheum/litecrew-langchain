## LiteCrew Monetization Strategy: Open-Core Pricing Tiers

### **LiteCrew Community (Free - MIT License)**
**Target:** Individual developers, startups, open-source projects

**Core Features:**
- Full agent orchestration framework
- Basic LLM provider support (OpenAI, Anthropic)
- Memory management (up to 1GB footprint)
- Standard error handling & recovery
- Community support via GitHub
- Basic CLI tools
- Core API access (current version)
- Documentation & tutorials
- Up to 3 concurrent agents
- Standard response times

**Why Free:** Drives adoption, builds community, establishes LiteCrew as the go-to lightweight alternative to CrewAI.
