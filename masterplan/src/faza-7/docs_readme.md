## 🚀 Quick Start

```bash
# Install
pip install litecrewai

# Create your first agent
litecrewai agent create "Research Assistant" \
  --model "gpt-4" \
  --tools "web_search,calculator,file_reader"

# Run a task
litecrewai task run "Find the latest AI news and summarize"
```

## 🎯 What is LiteCrewAI?

LiteCrewAI is a streamlined fork of CrewAI designed for personal use:

- **Lightweight**: <10MB memory per agent
- **Fast**: <100ms agent startup
- **Privacy-first**: No telemetry, local data
- **Cost-effective**: <$30/month infrastructure
- **Extensible**: Easy custom tools
